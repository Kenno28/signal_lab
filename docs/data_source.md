# Data Source

For V1, the project uses `yfinance` to download NVDA intraday OHLCV data.

## Selected Source

- Source: Stooq
- Ticker: NVDA
- Interval: 1D
- Period: from 1999

## Required Columns

- date
- open
- high
- low
- close
- volume

## Limitations


- Used only for learning and development
